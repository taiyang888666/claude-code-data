# A类：情绪×线香融合内容（60条）

## 一、情绪治愈系（第1-20条）

### 第1条 | 封面关键词：【焦虑】

```
今天刷了3小时手机，
啥也没干成，但焦虑感翻了3倍。

后来强迫自己放下手机，
点了一支沉香，盯着烟看。

【说实话前5分钟特别难受，脑子里全是没回的消息。】

但到第10分钟的时候，
突然就……安静了。

不是问题消失了，
是终于从那个焦虑的漩涡里爬出来了。

【你上一次什么都不做地待20分钟，是什么时候？】

今晚的沉香，适合放下手机的你。
```

### 第2条 | 封面关键词：【失眠】

```
凌晨2:47，又醒了。

数羊试过，白噪音试过，褪黑素也吃过。
脑子就是关不掉，像后台开了200个程序。

后来朋友给了我一盒线香，
说"你试试睡前点一支"。

第一次没啥感觉，
第二次闻着闻着居然睡着了。

【现在每晚睡前点一支薰衣草檀香，变成了一个仪式。】

闻到那个味道，身体就知道——
该休息了。

【你的"睡前仪式"是什么？评论区说说。】

今晚点的薰衣草檀香，适合所有睡不着的人。
```

### 第3条 | 封面关键词：【角落】

```
下班回到家，
在车里多坐了20分钟才上楼。

推开门的那一刻，
你是爸爸、是老公、是儿子，
唯独不是你自己。

【我现在每天给自己留20分钟，关上门，点一支香。】

这20分钟里，
不用回消息，不用假装开心，
不用对任何人负责。

就看着烟慢慢往上飘，
什么都不想。

【你有没有一个只属于自己的"角落"？】

今晚的崖柏，适合需要放空的你。
```

### 第4条 | 封面关键词：【崩溃】

```
昨天开会被骂了40分钟，
回到家坐在沙发上，一动不动。

老婆问我怎么了，我说"没事"。
其实眼眶已经红了，但就是哭不出来。

【成年人的崩溃是悄无声息的，不会大哭大闹，就是突然沉默了。】

后来她什么都没说，
点了一支檀香放在茶几上，
给我倒了杯热水就走了。

那个味道慢慢飘过来的时候，
我才终于把今天的委屈，
一点点放下了。

【你上一次在家人面前说"没事"，其实心里在崩溃，是什么时候？】

今晚的檀香，适合那些说不出口的疲惫。
```

### 第5条 | 封面关键词：【解压】

```
有人问我，压力大的时候怎么办？
别人喝酒，我点香。
至少第二天起来不头疼，
而且房间里还留着昨晚的余香。

今晚点的是沉香，适合解压。
```

### 第6条 | 封面关键词：【哭完以后】

```
哭完以后做什么？
洗把脸，点一支香，给自己倒杯热水。
然后告诉自己：
没事的，都会过去的。
能扛住的，就扛。扛不住的，就放下。

今晚点的是花香调，适合雨过天晴的心情。
```

### 第7条 | 封面关键词：【别说话】

```
情绪上来的时候，千万别说话。
说出去的话像泼出去的水，收不回来。
先点一支香，等烟散了再开口。
你会发现，很多话其实不用说。

今晚点的是檀香，适合冷静的时候。
```
