# 鸡汤×线香融合型IP —— 完整执行方案

> 核心理念：每一碗鸡汤都有一支香的味道，每一支香都带着一句让人安静下来的话
> 对标参考：宝仙悟语（鸡汤框架） × 亚卿倍儿香（线香专业） × 郑师傅（直播转化）
> 适用阶段：品牌初期、一人直播、全价格带

---

## 一、模式定义：什么是"鸡汤×线香融合型"？

### 三种模式对比

| 模式 | 代表账号 | 内容 | 优势 | 劣势 |
|------|---------|------|------|------|
| 纯鸡汤型 | 宝仙悟语（65万粉） | 人生道理+禅意语录 | 涨粉极快 | 粉丝不精准，挂产品转化低 |
| 纯卖货型 | 郑师傅（6万粉） | 产品卖点+价格促销 | 转化直接 | 没有情感粘性，留存差 |
| **融合型（你的路）** | **青澜沐·沉萦** | **人生感悟×点香场景×真实情绪** | **兼顾涨粉和转化** | **需要持续内容创意** |

### 融合型的核心公式

```
一条视频 = 一个情绪触发点 + 一个点香画面 + 一句走心的话 + 一个产品软植入

观众的心理路径：
  "这句话说到我心里了" → "他点的那支香好有氛围感" → "我也想要这样的20分钟" → 下单
```

### 为什么这个模式成立？

1. **线香天然=情绪产品**：没人因为"需要"才买线香，都是因为"想要一种感觉"
2. **鸡汤内容天然高传播**：情绪共鸣类视频的转发率是普通内容的2-4倍
3. **两者绑定后互相加强**：鸡汤让线香有灵魂，线香让鸡汤有"落地的道具"
4. **2026年情绪消费爆发**：香薰赛道增速36%+，"5分钟情绪急救"类产品付费意愿强

```

### 5. 统一封面模板

```
学宝仙的大字封面，但换成你的风格：

封面用统一模板（剪映/醒图都能做），只换文字
这样别人刷到你的主页，一眼就知道你是做什么的

```

【画面】固定机位，你坐在固定角落
        开头：手点燃线香（特写，2秒）

【文案结构】
  钩子（前3秒）：一句戳人的话
  正文（10-20秒）：展开情绪+场景描述
  收尾（3-5秒）："今晚点的是沉萦降真香，适合睡前安静的时候。晚安。"
                  （这句话就是软植入）

## 五、直播话术方案（第4周开始）

### 直播定位

```
不是"卖货直播"，是"陪伴式直播"

场景：你坐在固定角落，点着香，和大家聊天
节奏：慢的，安静的，像深夜电台
话术：鸡汤感悟 + 点香演示 + 自然推荐产品

核心感觉：观众进来像走进一个安静的房间，
         有一缕香飘着，有一个温柔的声音在说话
```

### 直播流程（每场1.5-2小时）

```
0-10分钟   → 【开场】点第一支香 + 今天的心情话题
10-30分钟  → 【聊天+种草】围绕一个情绪话题展开 + 介绍第一款香
30-40分钟  → 【逼单】福利价 + 消除顾虑
40-60分钟  → 【互动答疑】回答弹幕 + 主动提问
60-80分钟  → 【第二款产品】换一支香 + 换一个话题
80-90分钟  → 【收尾】今晚的话送给大家 + 预告明天
```

### 开场话术（逐字稿）

```
"晚上好，欢迎来到直播间。

 在这里，我们不急着买东西，
 先坐下来，安静一会儿。

 今天想跟大家聊一个话题：
 （抛出一个情绪话题，比如"你上一次什么都不想是什么时候？"）

## 八、数据指标与复盘

### 每日必看数据

| 指标 | 短视频及格线 | 直播及格线 | 低于及格怎么调 |
|------|-----------|-----------|-------------|
| 完播率 | >45% | — | 前3秒钩子不够，换更扎心的开头 |
| 点赞率 | >5% | — | 情绪共鸣不够强 |
| 评论率 | >1.5% | >3% | 结尾没引导互动 |
| 转发率 | >1% | — | 不够"有用"或不够"扎心" |
| 关注率 | >3% | >5% | 没给关注理由 |
| 停留时长 | — | >2分钟 | 内容不够吸引/节奏太快 |
| 商品点击率 | — | >8% | 产品过渡不自然 |
| 转化率 | — | >3% | 逼单力度不够/价格有问题 |

### 每周复盘清单

```
1. 本周哪条视频数据最好？→ 总结原因 → 下周多做这个方向
2. 本周哪条视频数据最差？→ 分析原因 → 停掉这个方向
3. 直播间停留时长变化？→ 是话术问题还是时段问题
4. 下单转化率变化？→ 是产品问题还是价格问题还是信任问题
5. 粉丝增长速度？→ 是否需要加大投流/增加更新频率
```
