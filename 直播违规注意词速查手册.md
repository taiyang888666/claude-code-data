## 一、极限词 / 绝对化用语（AI实时检测，触发即断播）

> 直播间语音AI检测准确率已超90%，方言谐音、拼音替代均可识别

| 类别 | 禁用词 | 安全替代 |
|------|-------|---------|
| 最高级 | 最好、最佳、最优、最强、最大、最便宜、最低价 | 非常好、很受欢迎、性价比很高 |
| 排名类 | 第一、全国第一、全网第一、销量第一、NO.1、Top1 | 销量领先、深受欢迎、热销款 |
| 唯一类 | 唯一、独一无二、独家、绝无仅有 | 特色款、我们家特有的 |
| 极端类 | 顶级、顶尖、极致、终极、天花板、鼻祖、王牌 | 精选品质、匠心之作、高品质 |
| 绝对承诺 | 100%、万能、永久、绝对、零风险、保证 | 效果因人而异、很多人反馈不错 |
| 权威虚构 | 国家级、世界级、权威认证（无证明）、专家推荐 | 通过检测、品质有保障 |
| 热梗滥用 | yyds、绝绝子（滥用时） | 真的很不错、我个人特别喜欢 |

## 二、健康功效词（线香类目核心雷区）

> 非医疗商品禁止使用医疗用语。这是线香直播最容易踩的坑。

### 禁用词一览

| 类别 | 禁用词 |
|------|-------|
| 睡眠 | 助眠、改善睡眠、促进睡眠、安眠、缓解失眠、治疗失眠、失眠克星、修复睡眠、深度睡眠、快速入眠、舒眠 |
| 安神 | 安神、养神、宁神、镇静、镇定、静心安神 |
| 情绪神经 | 舒缓焦虑、放松神经、缓解压力、减压、刺激神经 |
| 生理调节 | 调节神经、平衡内分泌、排毒、祛湿、调理身体 |
| 空气健康 | 净化空气、杀菌、消炎、除甲醛、抑菌、灭菌 |
| 脑力提升 | 醒脑、提神、提高专注力、增强记忆 |
| 功效断言 | 有效、可以治、能够改善、保证见效 |

### 安全替代话术

| 想表达的意思 | 安全说法 |
|------------|---------|
| 助眠 | "营造睡前仪式感" "适合睡前的味道" "让夜晚更温柔" |
| 安神 | "带来片刻宁静" "给忙碌的一天画个句号" "让人沉静下来" |
| 减压 | "回归平静的时刻" "偷得浮生半日闲" |
| 净化空气 | "带来清新的木质气息" "让空间更有层次感" |
| 提神醒脑 | "开启专注的仪式" "我自己感觉更容易静下来" |

**核心技巧：用"我觉得""我个人体验""很多人说"等主观句式，绝不用"可以""能够""有效"等断言句式**

## 三、封建迷信词（一级严重违规，可致永久封号）

| 类别 | 禁用词 |
|------|-------|
| 通灵神明 | 通神明、请神、通灵、与神沟通、神仙指点、财神到位 |
| 改运命运 | 改运、改命、转运、逢凶化吉、时来运转、万事亨通 |
| 辟邪镇宅 | 辟邪、镇宅、招财进宝、旺宅旺财、化解小人、旺夫旺子、挡灾 |
| 护身能量 | 护身、平衡正负能量、助吉避凶、转富招福、提升运势 |
| 宗教灵修 | 开光加持、算卦看命、六道轮回、灵验、有求必应 |

**安全说法：**
- 说"寓意美好" 不说"转运招财"

- 说"东方香道文化体验" 不说"通神明"
- 说"营造庄重的氛围" 不说"灵验/感应"
- 说"匠人手作、非遗工艺" 不说"开光加持"

## 四、虚假催单话术（2025-2026重点打击）

> 这是直播间特有的高频违规。平台对"催单话术"执行零容忍。

### 禁用表述

| 类别 | 禁用词/句 | 安全替代 |
|------|---------|---------|
| 虚假紧迫 | 最后一件了、再不抢就没了、马上涨价 | "库存有限" "今天活动价" |
| 虚假限时 | 仅此一次、最后一波、最后一天（无真实截止时间） | "这个价格今天有"（需真实） |
| 抢购营造 | 秒杀、抢爆了、万人疯抢、手慢无 | "欢迎下单体验" "先下单的先发货" |
| 恐慌诱导 | 错过就亏了、错过就没机会了、跳楼价 | "觉得合适就拍" "活动库存有限" |
| 虚假中奖 | 恭喜获奖、全民免单、点击有惊喜 | （直接不用） |
| 虚假催促 | 3、2、1上链接！倒数！快快快！ | "链接已经上了，需要的自己拍" |

### 虚假价格对比（高额罚款风险）

| 禁止行为 | 说明 | 处罚 |
|---------|------|------|
| 虚假划线价 | 展示虚高的"原价"制造大折扣假象 | 罚款 + 下架 + 扣保证金 |
| 先抬价后降价 | 先提高标价再发券，假装打折 | 交易额5%冻结30天 |
| 砍价剧本 | 表演砍价过程，原价是编的 | 严重可封号 |
| 虚假对比 | "别家卖XXX块"但数字是编的 | 罚款（案例：最高560万） |

**安全做法：**
- 价格对比只说模糊范围："同品质的大概在这个价位以上"
- 不编具体数字："别家卖288"（除非有真实截图证据）
- 折扣要真实：划线价必须是真实历史售价

## 六、引导私下交易 / 站外导流（严重违规，可致封号+罚款）

> 这是直播间最严重的违规之一。2025年4万+商家、5万+达人因此被处罚。

### 绝对禁止的行为

| 禁止行为 | 具体表现 |
|---------|---------|
| 口头导流 | "加微信""加v详聊""私信我""私下更优惠" |
| 谐音导流 | "薇信""卫星号""VX""威信" |
| 画面导流 | 屏幕上展示二维码、手机号、微信号 |
| 弹幕导流 | 助理/小号在弹幕刷联系方式 |
| 暗示导流 | "想要更多优惠的，你们知道怎么找到我" |

### 处罚

| 次数 | 处罚 |
|------|------|
| 首次 | 警告 + 流量降低50-80% + 禁言3-30天 |
| 再犯 | 扣健康分 + 收入冻结 + 关闭商品分享权限 |
| 严重 | 永久封号 + 高额罚款 + 关店 + 资金冻结 + 移送公安 |

## 七、贬低竞品 / 不正当竞争

### 禁用词

垃圾、智商税、假货、有毒、别买别人的、不如我们、碾压XX、吊打XX、秒杀同行

### 禁止行为

- 点名贬低："XX品牌太垃圾"
- 暗示贬低："别买隔壁那个"
- 价格羞辱："9块9能给你什么好东西？"
- 恶意对比：并排展示竞品负面画面

### 安全话术

- "每个品牌有每个品牌的特点，我只说我们的"
- "不同价位的线香差在哪"（不说"便宜的就是垃圾"）
- "我们用的是天然原料"（不说"别家用的是化学的"）

## 十、线香类目专用敏感词

> 以下词虽不全是"违禁词"，但在线香/香薰语境下容易触发审核

| 敏感词 | 风险 | 安全替代 |
|-------|------|---------|
| 化学香精 | 可能被认定暗示竞品有害 | "普通香""品质一般的" |
| 不含化学香精 | 可能被认定功效对比 | "天然植物配方" |
| 假沉香 | 涉嫌贬低行业 | "辨沉香""挑选好沉香""品质参差不齐" |
| 驱蚊 | 属于功效宣传（需农药登记证） | "带来清新的气息"（不提驱蚊功能） |
| 供佛/拜佛 | 本身不违规但要注意搭配 | 不说"灵验""有求必应"，说"营造庄重氛围" |
| 非遗传承人 | 需有真实资质证明 | 无证时说"传统手工""古法工艺" |
| 纯天然/零添加 | 需有检测报告支撑 | "天然原料""植物配方" |

## 十一、直播前30秒自查清单

每次开播前快速过一遍：

- [ ] 今天的话术里有没有极限词？（最、第一、唯一、顶级）
- [ ] 有没有功效断言？（助眠、安神、净化空气、杀菌）
- [ ] 有没有迷信词？（辟邪、转运、开光、招财）
- [ ] 价格对比是否真实？（划线价有依据吗？"别家卖XX"有证据吗？）
- [ ] 催单话术有没有虚假紧迫感？（最后一件、再不抢就没了）
- [ ] 有没有引导站外交易的内容？（加微信、私信）
- [ ] 有没有贬低竞品的话？（垃圾、假货、智商税）
- [ ] 所有功效描述是否用了主观句式？（"我觉得"而非"可以/能够"）
- [ ] 展示的商品和链接中的商品一致吗？
- [ ] 身份/资质表述真实吗？（非遗传承人需有证）

**全部通过 = 可以开播**

## 十二、处罚等级速览

| 等级 | 表现 | 处罚 |
|------|------|------|
| 轻微 | 画质差、短暂冷场、小口误 | 隐性限流10分钟-3天、提醒 |
| 一般 | 诱导关注、轻微功效词、服装不当 | 断播 + 禁播1-7天 + 扣健康分5-20分 |
| 中度 | 虚假价格、贬低竞品、诱导站外 | 关闭商品分享7-90天 + 收入冻结 |
| 严重 | 迷信、虚假身份、导流、录播 | 永久关闭电商权限 + 清退店铺 + 罚款 |
| 极严重 | 欺诈、色情、赌博 | 永久封号 + 资金冻结 + 移送公安 |

**健康分机制：**
- 满分100分
- 低于80分 = 部分功能受限
- 低于60分 = 直播间可能被关闭
- 一次严重违规可扣20分以上
