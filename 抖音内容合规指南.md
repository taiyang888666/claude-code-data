## 一、广告法极限词 / 绝对化用语（零容忍，触发即罚）

**法律依据：《广告法》第9条第3项**

### 违规词清单

| 类别 | 禁用词 |
|------|--------|
| 最高级 | 最、最好、最佳、最优、最强、最大、最低价 |
| 排名类 | 第一、全国第一、全网第一、销量第一、NO.1、Top1 |
| 唯一类 | 唯一、独一无二、独家、绝无仅有 |
| 极端类 | 顶级、顶尖、极致、终极、天花板、鼻祖、王牌 |
| 绝对承诺 | 100%有效、万能、永久、绝对、零风险 |
| 时间诱导 | 仅此一次、最后一波、再不抢就没了（无明确截止时间时） |
| 权威虚构 | 国家级、世界级、权威认证（无证明）、专家推荐（无真实专家） |
| 热梗变体（2026新增关注） | "yyds""绝绝子"滥用时也可能被标记 |

### 安全替代表达

| 违禁词 | 替代表达 |
|--------|---------|
| 最XX | 非常受欢迎、很多人推荐、热销款 |
| 第一/唯一 | 销量领先、深受欢迎 |
| 永久/100% | 长期使用效果更好、效果因人而异 |
| 顶级 | 精选品质、匠心之作 |

## 二、健康功效类违禁词（线香/香薰核心雷区）

**法律依据：《广告法》第17条 —— 非医疗商品禁止使用医疗用语或暗示治疗功能**

> 这是线香/香薰品类最容易踩的坑。平台AI+人工审核对以下词汇几乎必拦截。

### 违规词清单

| 类别 | 禁用词 |
|------|--------|
| 睡眠相关 | 助眠、改善睡眠、促进睡眠、舒眠、安眠、助入睡、缓解失眠、治疗失眠、失眠克星、修复睡眠、深度睡眠、快速入眠 |
| 安神相关 | 安神、养神、宁神、镇静、镇定 |
| 情绪/神经 | 舒缓焦虑、放松神经、缓解压力、减压（直接关联疾病时）、刺激神经 |
| 生理调节 | 调节神经、平衡内分泌、排毒、祛湿 |
| 其他功效 | 净化空气、杀菌、消炎、除甲醛、醒脑、提神、提高专注力 |
| 有害物质断言 | 释放甲醛、释放有害物质、含有苯（无检测依据时） |

### 安全替代表达

| 违禁表述 | 合规替代 |
|---------|---------|
| 助眠/改善睡眠 | 营造睡前仪式感、夜晚更安静的氛围、让夜晚更温柔、适合睡前的味道 |
| 安神/镇静 | 带来片刻宁静、陪你深呼吸、给忙碌一天画个温柔句号、让人放松/沉静 |
| 缓解压力/减压 | 回归内心的平静时刻、偷得浮生半日闲 |
| 净化空气/杀菌 | 带来清新自然的木质气息、让空间更有层次感 |
| 醒脑/提高专注力 | 开启专注的仪式、晨间清爽的唤醒感、我自己感觉更容易静下来 |
| 调理身心 | 日式/中式香道美学体验、东方禅意生活 |

**核心技巧**：
- 用"我觉得""很多人说""个人体验"等**主观句式**
- 避免"可以""能够""有效"等**断言句式**
- 重点落在"氛围""仪式感""情绪陪伴"，而非改变身体状态

## 四、对比贬低竞品（不正当竞争）

**2025年8月《违规营销：不正当竞争》实施细则生效，处罚升级**

### 违规行为

- 直接点名贬低："XX品牌太垃圾，我们这个更好"
- 间接暗示贬低："别买隔壁那个，成分差/智商税"
- 恶意对比画面：并排展示竞品负面 vs 自家正面
- 虚假打假/揭黑：实为踩竞品抬自家
- 价格羞辱："9块9能给你什么东西？只能是垃圾"

### 禁用词

垃圾、智商税、别买别人的、最差、不如我们、碾压XX、吊打XX、秒杀同行、假货、有毒

### 处罚梯度

| 等级 | 处罚 |
|------|------|
| 轻度 | 视频下架、限流7-15天 |
| 中度 | 关闭购物车/橱窗7-90天 |
| 重度 | 永久关闭电商权限、清退店铺、冻结佣金3年 |

### 安全表达方式

- 只说自家优势，不提竞品名称
- 如需对比，用"市面常见XX vs 我们的"等中性表述，且有数据支撑
- "每个品牌有每个品牌的特点，我只说我们的"
- "不同价位的线香差在哪" 而非 "9块9的线香是什么垃圾"

## 五、恐吓营销类违禁词

### 违规行为

通过制造恐惧、夸大危害来促使消费者购买

### 禁用表述

| 违禁表述 | 合规替代 |
|---------|---------|
| 你点的香可能有毒 | 不同线香闻着差别有多大 |
| 释放的是甲醛、苯 | 跟原料和配方有关系 |
| 化学香精含量过高，刺激神经 | 闻起来可能会觉得不太舒服 |
| 含有麝香，闻多了会上瘾甚至伤身 | （直接删除，无法安全替代） |
| 垃圾原料 | 原料不行 / 原料品质一般 |

## 六、其他敏感表述

### "化学香精"相关

"化学香精"这个词本身不算违禁词，但容易引发以下风险：
- 暗示竞品"有毒""有害" → 不正当竞争
- "不含化学香精" → 可能被认定为暗示功效对比

**安全替代**：
- "化学香精" → "普通香" 或 "品质一般的"
- "不含化学香精" → "天然植物配方" "天然原料"
- "化学香精的太假了" → "品质一般的，颜色太均匀了"

### "假沉香"相关

- "八成不是真沉香" → "品质差异确实很大"
- "假沉香" → 改用"辨沉香""挑选好沉香"
- "假货" → "品质参差不齐"

## 八、发布前自查清单（30秒快速检查）

每条短视频/直播话术发布前，逐条检查：

- [ ] **功效词检查**：文案中是否出现"改善""治疗""缓解""杀菌""驱""镇定""助眠""安神"等词？
- [ ] **句式检查**：表述是否为"我觉得""个人体验"等主观句式，而非"可以""能够""有效"等断言？
