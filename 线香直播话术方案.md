# 线香直播话术方案（一人直播版）

> 账号：青澜沐·沉萦 | 品牌：沉萦
> 针对场景：品牌初期、主播一人扛、互动冷清、全价格带产品

---

## 一、首先解决：品牌人设定位（开播前必须想清楚）

你需要先给自己一个"记忆点"，观众才有理由停留。参考方向（选一个最贴合你的）：

| 人设方向 | 关键词 | 适合的人 |
|---------|--------|---------|
| 产地匠人型 | "我们家三代做香" / "从原料开始把控" | 有真实产地、工厂资源 |
| 文化讲师型 | "教你闻香识香" / "中式生活美学" | 有一定香道知识储备 |
| 真诚测评型 | "我自己点过上百种香" / "只推我自己用的" | 真实用户视角、个人魅力型 |
| 性价比工厂型 | "工厂直发，没有中间商" / "同品质别家卖一两百" | 有供应链价格优势 |

**建议**：刚起步最容易上手的是 **"真诚测评型"** —— 不需要编故事，就是真实分享自己用香的感受，门槛最低，可信度高。

---

## 二、直播话术全流程框架

### 📋 整体节奏（每30分钟一个循环）

```
0-5分钟   → 【开场暖场】拉停留
5-15分钟  → 【产品讲解】塑价值
15-20分钟 → 【逼单转化】促下单
20-25分钟 → 【互动答疑】拉互动数据
25-30分钟 → 【福利预告】拉留存，进入下一轮
```

---

### 第1环节：开场暖场（0-5分钟）

**目标**：让刷到的人停下来，不要划走

#### 话术模板：

```
"家人们晚上好，欢迎来到沉萦直播间~
我是青澜沐，沉萦品牌的主理人。
今天给大家点一支我最近特别喜欢的香——
你们闻不到，但我形容给你们听——
这支香前调是清甜的花果香，中调慢慢变成沉稳的木质香，
尾韵带一点点奶甜，特别适合睡前点。
喜欢线香的、家里有佛堂的、喜欢品茶的，都可以停下来看看。
新来的朋友点个关注，今天有福利款，价格只在直播间有。"
```

#### 要点：
- **不要一上来就卖货**，先"点香+描述气味"，这是线香类目独有的内容优势
- 用通感描述（视觉、味觉类比）代替观众闻不到的嗅觉
- 明确告诉观众"谁该留下来"——筛选精准用户

#### 冷场自救话术（没人互动时用）：

```
"刚进来的家人们，如果你家里经常点香的，打个"点过"，
我看看咱们直播间有多少懂香的朋友~"

"新来的朋友可能不了解我，我是青澜沐，沉萦的主理人。
不是什么大主播，就是自己真的喜欢线香，点了几百种了，
今天把我觉得真正好的分享给你们。"
```

---

### 第2环节：产品讲解——塑造价值（5-15分钟）

**目标**：让观众觉得"这个东西值这个价"

#### 话术框架：五步价值塑造法

**第①步：场景代入**（让观众想象自己用的画面）

```
"你想象一下，晚上下班回到家，泡一杯茶，
点上一支沉香，那个烟丝慢慢升起来，整个房间都安静下来了——
这种感觉，是任何香水、香薰机都给不了你的。"

"家里供佛的家人们，好香和差香的区别太大了，
差的香点完满屋子呛嗓子，头疼，
好的天然香，点完你会觉得整个人都沉静下来。"
```

**第②步：原料/工艺讲解**（建立专业感）

```
"这支香的原料是海南沉香粉+天然植物粘粉，
天然原料和普通原料的区别在哪？
你看——（展示香灰/烟丝）
天然香的烟丝是细的、直的，灰是白色偏灰的，不会发黑卷曲。
品质一般的你一闻就能感觉到区别，味道比较冲。"

"我们这个是天然植物粉+天然粘粉，没有化学粘合剂。
你收到可以自己验——掰开看截面，纹理是均匀的，
不是那种光滑的工业批量压出来的。"
```

**第③步：对比打击**（不点名，但让观众有参照）

```
"说实话，线香这个行业水比较深。
市面上特别便宜的那种，大家自己想想原料成本够不够。
光是天然沉香粉的原料成本，一克就要好几块钱，
所以价格太低的，品质上多少会打折扣。

我不是说贵的就一定好，但好的一定不会太便宜。
我们这个定价，就是把中间商利润砍掉，直接给到大家。"
```

**第④步：信任凭证**（拿出证据）

```
"不是我嘴上说天然就天然的——
（展示检测报告/原料照片/制作过程视频）
这是我们的检测报告，各项指标都达标，
你们可以截图，收到货自己对比验证。

老粉都知道，我们家退货率很低，
因为收到你一闻就知道，好东西是骗不了鼻子的。"
```

**第⑤步：价格锚定**（塑造性价比）

```
"这个品质的线香，你们去香道馆买，一盒最少一两百块。
我之前在某某品牌看到类似的，标价两三百。
我们今天直播间的价格——等下公布，你们先猜猜~
（停顿3-5秒制造悬念）
我先把其他的介绍完，价格最后说。"
```

---

### 第3环节：逼单转化（15-20分钟）

**目标**：把心动变成行动

#### 话术模板：

**① 价格揭晓 + 限时限量**

```
"好，价格来了——
这一款沉萦古法降真香，外面正常卖六七十块，
今天直播间，券后价只要29块9。
但这个价格只有今天有，我备货就这么多，
卖完就恢复日常价了。"
```

**② 下单播报（一人直播版，自己念）**

```
"好的，看到有人拍了——谢谢这位家人的支持！
放心，我给你安排优先发货。
还没拍的抓紧，库存我看下——还有不多了。"

（注意：即使下单少也要说，营造有人买的氛围）
（如果确实没人买，就说"我看后台有人加购了还没付款，
犹豫的家人不用担心，我们支持7天无理由退，不喜欢退回来就行"）
```

**③ 消除顾虑**

```
"第一次买的家人，不用担心。
运费险有的，7天无理由退。
你收到货先闻，不好闻直接退，我们承担来回运费。
我敢这么说，就是因为退货率很低——因为东西是真的好。"
```

**④ 从众引导**

```
"老粉都知道，这款是我们家复购率最高的，
好多人一次买三四盒，自己用一盒，送人两三盒。
线香送人特别有面子，包装也好看，
比送烟送酒有品位多了。"
```

---

### 第4环节：互动答疑（20-25分钟）

**目标**：拉互动数据（评论、点赞），让算法推更多流量

#### 主动提问话术（不要等观众问，主动抛问题）：

```
"家人们，你们平时在家点香吗？点什么香？
扣'点过'或者'没点过'，我看看~"

"你们买线香最怕什么？
我猜——担心买到品质不好的对不对？有同感的打个'有'"

"给大家出个小知识——
你们知道沉香、檀香、崖柏，哪个特别适合睡前点吗？
知道的扣答案，不知道的扣个问号，我来给大家讲。"

"你们家里是什么场景点香？
书房打'书房'，客厅打'客厅'，卧室打'卧室'，佛堂打'佛堂'"
```

#### 回答常见问题的标准话术：

| 常见问题 | 话术模板 |
|---------|---------|
| "会不会呛" | "好的天然香闻着是柔和的，不会让你不舒服。我们这个你收到打开闻一下就知道，凉凉的、清甜的，很舒服。" |
| "能烧多久" | "这一支大概能燃30-40分钟，一盒20支左右，一天点一支能用大半个月。算下来一天才几毛钱，比奶茶便宜多了。" |
| "和XX品牌比怎么样" | "每个品牌有每个品牌的特点，我只说我们的——我们用的是海南天然沉香原料，不加香精，你收到对比一下就知道区别了。" |
| "有没有更便宜的" | "有的，我们有入门款9块9的松塔香，新朋友可以先从入门款试，觉得好再回来买进阶款。" |

---

### 第5环节：福利预告 + 循环过渡（25-30分钟）

```
"好，刚才那款先到这里，喜欢的赶紧拍，库存有限。
接下来我给大家介绍另一款——这款更特别，
等下价格也会让你们惊喜，先别走啊。

刚进来的新朋友，点个关注，等下有一个福利款，
价格我先不说，反正你们点了关注不亏。"
```

## 三、一人直播专用——冷场急救话术包

你一个人播，没有助理刷屏、没有粉丝团配合，最大的挑战就是冷场。
以下是不同冷场情况的应对：

### 情况1：在线人数极少（<10人）

```
"没关系，人少我讲得更仔细。
其实人少的时候我反而更放松，
像朋友之间聊天一样，给你们慢慢讲。
人少福利大——今天在的都是有缘人。"
```

### 情况2：没人说话/没弹幕

```
"我看在线有X个人，你们都在默默听是吧？没关系。
我理解，很多人刷到直播间就是先静静看看，
觉得好再互动，这很正常。
我继续讲，你们有问题随时打在公屏上。"

（然后自然地转回产品讲解，不要反复催互动，会显得急切）
```

### 情况3：没人下单

```
"没关系，第一次买线香确实会犹豫，
毕竟闻不到嘛。但是我们有7天无理由退，
你可以先拍一个入门款试试，
不好闻退回来，来回运费都不用你出。
就当花几十块钱验证一下我说的是不是真的。"
```

### 情况4：有人问奇怪的问题/抬杠

```
"感谢这位朋友的问题~每个人对香的理解不一样，
我只分享我自己的真实体验。
咱们求同存异，都是爱香的人。"

（不争论、不解释太多，简单回应后立刻转回产品话题）
```

## 四、产品上架节奏建议（全价格带排列）

一人直播精力有限，建议每场控制在 **3-5个品**，按以下节奏安排：

```
开场第1个品 → 【引流福利款】沉萦松塔香 ¥9.90 / 古法合香线香50克 ¥9.90起
               目的：拉停留、破零单、练手
               话术重点：性价比、试错成本低、"不到一杯奶茶的钱"

第2个品     → 【主推利润款】沉萦古法降真香 ¥29.90 / 鹅梨古法合香 ¥19.90~¥178
               目的：主要出单品，利润来源
               话术重点：价值塑造、品质对比、多规格可选

第3个品     → 【高端品鉴款】观沙无粘粉线香 ¥88 / 沉香香薰礼盒 ¥499 / 棋楠小马 ¥88.80~¥288
               目的：拉客单价、筛选高净值用户
               话术重点：稀缺原料、文化故事、送礼场景

第4个品     → 【返场福利款】天然降真香椰奶桂花 ¥18.80起 / 古法合香线香 ¥19.90起
               目的：收尾冲单量、拉互动
               话术重点：限时限量、最后机会

（循环回到第1个品，开始新一轮）
```

## 五、开播前准备清单

### 场景布置
- [ ] 背景干净整洁，最好有中式元素（茶具、书法、木质桌面）
- [ ] 准备一个香插/香炉，开播就点上一支（有烟丝飘动的画面特别吸引人停留）
- [ ] 灯光柔和暖色调，不要大白光
- [ ] 准备手持小卡片/产品对比样品，方便展示

### 产品准备
- [ ] 每个品准备一支已经点燃到一半的（展示香灰颜色和形态）
- [ ] 准备一支对比样品，展示不同品质的差异
- [ ] 检测报告/质检单打印出来，随时可以举起来给镜头看

### 话术准备
- [ ] 把每个品的核心卖点写在手边的小纸条上（3个关键词就够）
- [ ] 准备5个主动提问的问题（防冷场）
- [ ] 记住价格锚点数字（别人卖多少、你卖多少、省了多少）

## 七、数据复盘——每场直播后必看

| 指标 | 关注点 |
|------|--------|
| 平均停留时长 | 低于1分钟 → 开场话术有问题，不够吸引人 |
| 互动率（评论/点赞） | 低 → 主动提问不够，话术太单向输出 |
| 商品点击率 | 低 → 产品讲解没塑造出价值，观众没兴趣看 |
| 点击→下单转化率 | 低 → 逼单话术或价格/信任有问题 |
| 粉丝团转化 | 低 → 没有引导加粉丝团的话术 |
