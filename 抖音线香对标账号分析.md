## 二、重点对标账号详细拆解（建议你逐个去看）

### TOP 1：千香百集（96万粉）—— 学人设故事

```
IP标签：80后非遗制香师 | 海南移居创业
为什么火：
  - 有完整的人生故事线："80后放弃XX，全家移居海南做香"
  - 从"个人选择"到"家族事业"的叙事弧
  - 内容 = 制香日常 + 海南生活 + 香道知识
你能学的：
  → 找到你自己和"香"的故事（为什么开始做香？什么契机？）
  → 把这个故事反复讲，做成你的标签
```

### TOP 2：沉香女儿珍藏阁（22万粉）—— 学直播节奏

```
IP标签：沉香之乡源头商家 | 每天早上6:40开播
为什么火：
  - 固定时间开播（6:40），养成用户习惯
  - "女儿"人设 = 年轻 + 亲切 + 家族背景
  - 带动产业带爆发（工厂扩大10倍、种植5万亩）
你能学的：
  → 固定开播时间，让粉丝形成观看习惯
  → 产地实拍内容（即使你不在产地，去采购时拍素材）
```
