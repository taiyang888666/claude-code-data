## 用户偏好与流程
- **所有公众号操作都由 Claude 通过浏览器自动化完成**（用户不手动操作后台）
- 公众号发布形式：**贴图**（图片轮播），不是文章
- 只保存草稿，不直接发布
- 不要用API的 `draft/add` + `freepublish/submit`，那样会发成文章格式
- **已验证可行的完整流程（浏览器自动化）**：
  1. 运行 `公众号图片号生成器_v4.py` 生成图片到 `公众号图片/` 目录
  2. 通过微信API上传图片到素材库（`add_material`接口）
  3. 浏览器打开贴图编辑器（URL参数：`t=media/appmsg_edit_v2&action=edit&isNew=1&type=77&createType=8`）
  4. **逐张**按正确顺序选择图片（批量选择会按grid显示顺序添加，不是点击顺序）
  5. 填写标题（ProseMirror div，index 0，max 20字）
  6. 填写描述（ProseMirror div，index 1）+ 末尾加 #话题标签
  7. 选择系统推荐话题（`.js_recommend_topic` 面板，可能需要 `display:block` 激活）
  8. 点击"保存为草稿"
- **技能文件**：`~/.claude/skills/publish-tietie.md`
- **关键技术细节**：
  - 图片选择器中图片按上传时间倒序排列（最新在前）
  - 必须**逐张添加**以保证正确顺序
  - ProseMirror 输入：修改 `innerHTML` + 触发 `input` 事件，不能用 `value`
  - 直接上传（CDP file upload）不可行，微信上传组件有特殊验证
