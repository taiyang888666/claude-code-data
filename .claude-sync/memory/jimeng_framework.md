# 即梦AI视频提示词框架

来源文件：D:\桌面\即梦提示词(1).xlsx

## 完整框架（24步）

| 序号 | 步骤 | 说明 |
|------|------|------|
| 1 | 全局画面框架 | 画面比例，如9:16 |
| 2 | 什么类型的视频 | 如：抖音短视频 |
| 3 | 竖版or横版 | 如：竖屏 |
| 4 | 用什么设备去拍 | 如：iPhone后置原相机 |
| 5 | 整体风格 | 设备+实拍感+品类风格组合描述 |
| 6 | 关键点 | 拒绝过度磨皮的AI塑料感、颗粒感、塑料感 |
| 7 | 打光效果 | 如：自然温暖的生活化打光 |
| 8 | 清晰度 | 极高清晰度 |
| 9 | 视频画面重点是谁 | 核心视觉焦点，放大产品质感 |
| 10 | 剪辑节奏 | 快/慢/卡点节奏 |
| 11 | BGM | 背景音乐风格描述 |
| 12 | 画面动作配合 | 配合什么动作节奏 |
| 13 | 画面颗粒度细节 | 微观细节描述（音效/视觉） |
| 14 | 画面内人物设定 | 年龄、人设定位 |
| 15 | 人物衣着特写 | 穿着风格描述 |
| 16 | 人物行为特点样貌特点 | 面部/气质描述，保留真实皮肤纹理 |
| 17 | 关键点：拒绝高度美颜导致AI味 | 无夸张美颜滤镜 |
| 18 | 人物动作 | 动作风格描述 |
| 19 | 人物行为 | 核心行为的感染力描述 |
| 20 | 精神面貌 | 眼神/情绪状态 |
| 21 | 分镜组合 | 衔接说明，无需填文本，用于理解结构 |
| 22 | 镜头N | 镜头编号+内容主题+时间段 |
| 23 | 镜头N的文案 | 含：讲话语感提示词 + 音色提示词 + 具体台词 |
| 24 | 镜头N的AI提示词 | 含以下子项（每项单独描述）：画面提示词、场景提示词、动作提示词、产品提示词、灯光提示词、细节提示词（细致）、运镜提示词（细致）、字幕/花字提示词 |

## 分镜规则
- 开头建议3秒
- 后续每个分镜按5秒内容填充
- 分镜数量决定视频长短
- 每个分镜重复22-24步骤

## 使用要点
- 步骤1-20为全局设定，每个视频填一次
- 步骤22-24为分镜模块，每个镜头重复填写
- 第6、17步反复强调"去AI感"，是即梦出片质量的关键
- 第24步的AI提示词是最核心的部分，需要拆分为：画面/场景/动作/产品/灯光/细节/运镜/字幕 八个维度逐一描写
