## 学到的设计规律

### 标题格式
- 统一格式：`配色分享｜[主题]配色`（使用全角竖线分隔）
- 简洁明了，直接告诉用户内容主题
- 可借鉴到公众号贴图：`[主题关键词]｜[吸引点]`

### 标签策略（每帖9-10个）
- 大类标签：#配色灵感 #配色分享 #颜色搭配 #色彩搭配
- 品质标签：#高级感配色 #审美提升 #色彩美学
- 资源库标签：#色彩灵感库
- 主题特定标签：#港式复古 #多巴胺配色 等
- 原创标签：#oc

### 配色风格趋势（可用于公众号贴图设计）
1. **复古风** — 最受欢迎，包括港式复古、美式复古
   - 特点：低饱和度、暖色调、怀旧感
   - 常用色：暗红、深棕、橄榄绿、暗金
2. **多巴胺风** — 适合夏季/活力主题
   - 特点：高饱和、明亮、对比强烈
   - 常用色：亮粉、荧光绿、明黄、电蓝
3. **暗黑系** — 适合深夜/情绪主题
   - 特点：深色背景、冷色调、神秘感
   - 常用色：深紫、暗蓝、碳黑、暗红
4. **二次元/动漫风** — 年轻受众
   - 特点：清新、渐变、柔和
5. **反面教材** — "超难受的配色" 等内容反向流量

### 对公众号贴图的借鉴
- 封面图设计可参考复古风/暗黑系色调，营造高级感
- 内容图可运用舒适的配色组合，提升阅读体验
- 标题用简洁格式 + 标签引流
- 系列化运营：建立固定的视觉风格/色系

## 应用到"青澜传媒"贴图的建议
- **焦虑主题** → 适合暗黑系 + 温暖过渡色（深蓝→暖橙）
- **失眠主题** → 适合暗蓝/深紫色调，配以柔和的月光色
- **角落主题** → 适合复古风暖色调，营造温馨安静感

### 排版核心原则
1. **一致性** — 同一系列贴图保持统一的字体、配色、版式
2. **留白** — 不要把图片填满，适当留白显高级
3. **层级清晰** — 标题 > 副标题 > 正文 > 注释，字号逐级递减
4. **对齐** — 文字和元素严格对齐（左对齐或居中），不混用
5. **色块分区** — 用色块区分不同内容区域，引导视线

### 4张图的排版建议（1080x1440竖版）
- **01_封面**：大标题 + 主视觉元素，配色鲜明，一眼抓人
- **02_内容**：核心信息排版，图文结合，字号适中可读
- **03_产品**：产品展示为主，简洁背景，重点突出
- **04_尾页**：品牌logo + 引导关注 + 联系方式，简洁收尾
